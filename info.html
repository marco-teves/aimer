aimr - how to play

Enemy Types

Standard

  • Scoring an enemy of this type will spawn another enemy randomy on the screen.
  • Players should be aware of the screen in order to keep up with them
  • Gives +1 point

Feinter

  • This enemy type will spawn like a standard enemy, except that it will fient to a different location quickly.
  • Players wil need good awareness and flicking ability to be able to score this enemy.
